CLOPS — Command Line OPtionS (BETA)

Given a description of command-line options, in a custom language, CLOPS generates a Java command-line parser and documentation. The user can explicitly express dependencies between the individual options such as exclusion or overrides. Take a look at how it compares to related projects.
